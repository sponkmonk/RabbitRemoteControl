Detailed Description

Connect interface.

It is implemented by the protol plugin. It starts a timer by default to start a non-Qt event loop (that is, normal loop processing) . See Connect(), slotTimeOut(), OnProcess() for details. Of course, it still supports Qt events (the signal-slot mechanism of QObject).

Note
  • The interface only is implemented by plug-in.
  • Its instance is in a background thread.
  • The plug-in needs to implement the following interface.
    • OnInit()
    • OnClean()
    • OnProcess()
    • slotClipBoardChanged()
