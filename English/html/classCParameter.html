Detailed Description

The CParameter interface class.

The parameter has the following type:

  1. Only valid in the plugin. The application cannot access it directly, it can only be set via CConnecter::OpenDialogSettings.
    See also
    CParameterConnecter
  2. Valid in the Client and the plugin. The application cannot access it directly, it can only be set via CClient::GetSettingsWidgets. Plugins can use them directly or use them as initialization values. For example: save the password can use it as the initialization value. See: CConnecter::SetParameterClient
    See also
    CParameterClient
  3. Valid in both the application and the Client and the plugin. This type is not used at this time.
